# Agência Autônoma

Sistema automatizado de prospecção para clínicas de estética.

## Objetivo
Encontrar clínicas sem site, gerar abordagem personalizada com IA e criar protótipos rápidos de landing pages.

## Stack
- Apify
- Make.com
- Claude API
- Lovable
- Instantly
- Telegram

## Fluxo
Google Maps → IA → Protótipo → Aprovação → Disparo
